**Desarrollo proyecto final - máster Desarrollo Web FullStack con el entorno MERN.**

Código fuente en Javascript usando el framework Express para la parte del Backend (API) donde se implementó:

- Seguridad Login 
- Registro
- Login social (Google)
- Configuración de preferencias de usuario
- Cambio de clave
- Gestión de eventos usando el API de TICKETMASTER
- Guardar eventos a favoritos
- Eliminar eventos de favoritos
- Listar eventos favoritos
- Cierre de sesión

Se utilizó base de datos MONGO.
